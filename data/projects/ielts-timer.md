## IELTS Timer Application

The IELTS Timer is a user-friendly tool designed to help users practice timed sections of the IELTS exam. With customizable durations and an automatic stage progression feature, this timer enables users to simulate real test conditions efficiently. 

### Key Features
- **Simple Start and Pause**: Start the timer with a single click for a streamlined user experience.
- **Customizable Durations**: Easily adjust each stage's duration through the settings panel to match personal practice needs.
- **Automatic Stage Transition**: When a stage ends, the timer seamlessly moves to the next stage, mimicking the continuous flow of the actual IELTS exam.

This IELTS Timer is an ideal resource for candidates preparing for the IELTS exam, allowing them to effectively manage time and improve pacing.
